# Segurança

## Credenciais

- **Nunca** faça commit do ficheiro `.env` nem do token do bot.
- O repositório inclui `.gitignore` para ignorar `.env`; confirme antes de `git push`.
- Se um token vazar, **revogue imediatamente** no [Discord Developer Portal](https://discord.com/developers/applications) e gere um novo.

## Reportar problemas

Se encontrar uma vulnerabilidade de segurança neste projeto, contacte o mantenedor de forma privada (por exemplo, mensagem no GitHub com visibilidade restrita, se disponível) em vez de abrir um issue público com detalhes exploráveis.

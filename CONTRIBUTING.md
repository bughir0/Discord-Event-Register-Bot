# Contribuir

Obrigado pelo interesse em melhorar este bot.

## Como contribuir

1. Faça um **fork** do repositório.
2. Crie um branch para a sua alteração (`git checkout -b minha-feature`).
3. Mantenha o código em **TypeScript** consistente com o estilo existente.
4. Execute `npm run lint` (verificação TypeScript) antes de submeter.
5. Abra um **Pull Request** com descrição clara do que mudou e porquê.

## O que evitar

- Incluir tokens, URLs com senha ou ficheiros `.env` reais.
- Alterações grandes sem contexto; prefira PRs focados.
